# 객체지향 프로그래밍
> Object-oriented programming (OOP)

프로그램을 객체 간의 상호작용으로 나타내는 프로그래밍 방식

특징
- 높은 재사용성 (상속)
- 유지보수 용이 (객체 관계, 모듈화)
- 높은 보안성 및 신뢰성 (캡슐화, 중복 제거)
